# Seed Changer

Dev tool that allows game seed to be changed mid-run.

To use it: press F2 at the Map screen, enter your new seed and then reload the save.

Please do not use this for cheating :(

## Installation

Install [BepInEx](https://across-the-obelisk.thunderstore.io/package/BepInEx/BepInExPack_AcrossTheObelisk/).

Download the [latest release](https://github.com/stiffmeds/Seed_Changer) and put it in your BepInEx _plugins_ folder.
